pet travel finder

A responsive front-end basic web app designed to help users explore pet-friendly travel experiences and destinations. Built with HTML, CSS, and JavaScript on CodePen, this project combines modern UI design with a travel-focused user experience for pet owners. It needs an API added in the near future for real results. Please undersatnd this is just front end pure code. To practice.
